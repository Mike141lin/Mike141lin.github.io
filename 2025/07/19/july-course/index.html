7月—cs144、cs106L、Computer Networking: A Top-Down Approach

Cs144

基础概念 (第一周):

•数据报（Datagrams）
•封装（Encapsulation）
•多路复用（Multiplexing）

传输层 (第二、三周):

•如何在不可靠的网络上实现可靠传输，重点讲解TCP协议的实现。

网络层 (第四至八周):

•包交换（Packet Switching）
•拥塞控制（Congestion Control）
•网络地址转换（NAT Traversal）
•路由（Routing）算法。

高级主题 (第九、十周):

•弹性缓冲区（Elastic Buffers）
•网络安全（Security）。

Computer Networking: A Top-Down Approach

第1章： 概述 (Chapter 1: Introduction)
第2章： 应用层 (Chapter 2: Application Layer)
第3章： 传输层 (Chapter 3: Transport Layer)
第4章： 网络数据平面 (Chapter 4: Network Data Plane)
第5章： 网络控制平面 (Chapter 5: Network Control Plane)
第6章： 链路层和局域网 (Chapter 6: Link Layer and LANs)
第7章： 无线与移动网络 (Chapter 7: Wireless and Mobile Networks)
第8章： 网络安全 (Chapter 8: Network Security)

CS106L

第一部分：C++核心基础 (C++ Core Fundamentals)

•课程介绍 (Welcome!)
•类型与结构体 (Types and Structs)
•初始化与引用 (Initialization and References)
•输入/输出流 (Streams)

第二部分：数据处理与访问 (Data Handling and Access)

•容器 (Containers)
•迭代器与指针 (Iterators and Pointers)

第三部分：面向对象编程 (Object-Oriented Programming)

•类 (Classes)
•继承 (Inheritance)
•运算符重载 (Operator Overloading)
•特殊成员函数 (Special Member Functions)

第四部分：泛型编程与现代特性 (Generic Programming & Modern Features)

•常量正确性与类模板 (Const Correctness & Class Templates)
•函数模板 (Function Templates)
•函数与Lambda表达式 (Functions and Lambdas)

第五部分：高级内存管理与项目实践 (Advanced Memory Management & Practice)

•移动语义 (Move Semantics)
•std::optional与类型安全 (std::optional and Type Safety)
•RAII、智能指针与项目构建 (RAII, Smart Pointers, and Building C++ Projects)
