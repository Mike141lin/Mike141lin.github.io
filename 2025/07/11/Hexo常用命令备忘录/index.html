Hexo & Git 常用命令备忘录

欢迎来到我的新博客！

在搭建这个小站的过程中，我遇到了不少问题，也学到了很多新的终端命令。为了防止将来遗忘，也为了给像我一样的新手朋友提供一个参考，我决定把这些最核心、最常用的命令整理下来，作为我的第一篇文章。

希望这份备忘录能成为你在博客世界里探险的实用地图。

Hexo 核心工作流

这是我们日常与博客打交道最常用的一套命令，覆盖了从创建、写作到发布的全过程。

hexo new：创建新内容

这是我们写作的起点。

# 创建一篇新的文章，文件会生成在 source/_posts/ 目录下
hexo new "我的第一篇文章"

# 创建一个独立的页面，比如“关于我”
hexo new page "about"
